## Development server
1. git clone 
2. cd /frontendsr
3. Run `npm install` for install dependencies. 
4. Run `ng serve` for a dev server. 
5. Navigate to `http://localhost:4200/`. The app will automatically reload  

## Tests

Run `ng test` to execute the unit tests via Karma.
